1) взять rapier2d + pixi.js
2) сгенерить простенькую сцену из статических коллайдеров
3) закинуть катающуюся голову дрона + закрепленный на него "инструмент для добычи ресурсов"
4) прикрепить к голове источник света
5) "осветить" ближайшие коллайдеры по принципу "если на это напрямую попадает луч, то освещаем это относительно расстояния"
